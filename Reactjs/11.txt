📘 What is Babel and What Exactly Does It Do?

✅ Babel is a JavaScript compiler (transpiler). It performs **two main jobs**:

──────────────────────────────
🔧 1. Convert Modern JavaScript (ES6+) ➝ Older JavaScript
──────────────────────────────
👉 Browsers (especially old ones) don’t understand modern features like:
- let, const
- arrow functions
- async/await
- optional chaining (?.)
- spread/rest operators
- import/export modules

✅ Babel rewrites this modern code into old JavaScript so that it runs on all browsers.

──────────────────────────────
💡 2. Convert JSX ➝ JavaScript
──────────────────────────────
