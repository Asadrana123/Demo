🛠️ Common Babel Presets and Plugins

1. Presets (groups of plugins):

- @babel/preset-env
  • Transforms modern JavaScript (ES6+) into backwards compatible JS based on target browsers.

- @babel/preset-react
  • Transforms JSX and React-specific syntax into JavaScript.

- @babel/preset-typescript
  • Transforms TypeScript code into JavaScript.

2. Popular Plugins (single transformations):

- @babel/plugin-transform-arrow-functions
  • Converts arrow functions ( ()=>{} ) into regular functions.

- @babel/plugin-transform-async-to-generator
  • Transforms async/await into generator functions for compatibility.

- @babel/plugin-proposal-class-properties
  • Enables support for class properties syntax in JavaScript classes.

- @babel/plugin-transform-runtime
  • Optimizes helper code to avoid duplication and reduce bundle size.

- @babel/plugin-proposal-optional-chaining
  • Adds support for optional chaining operator (?.).

- @babel/plugin-proposal-nullish-coalescing-operator
  • Adds support for ?? operator.

3. Custom plugins can also be created for specific transformations.

📌 Note:
Presets bundle many plugins together to simplify configuration.
