1. type — what can it be?

String — HTML tags like 'div', 'h1', 'span', etc.
Function — React functional component (e.g. MyComponent)
Class — React class component (extends React.Component)

2. props — what can it be?
Object — with any valid props/attributes you want to set
Example: { id: 'title', className: 'header', onClick: () => {} }
Can be null or undefined if no props

3. children — what can it be?
Strings or numbers — text content inside the element
React elements — nested elements or components
Arrays — list of children
null, undefined, or false — ignored, no rendering
Note: Functions cannot be passed as children (unless as a component)
