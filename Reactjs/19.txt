1. What problem does create-react-app solve?
Before CRA, if you wanted to build a React app, you had to manually:

Install react, react-dom
Set up Webpack to bundle your files
Set up Babel to convert JSX into JS
Set up ESLint to check your code
Create scripts for npm start, npm build, etc.
Configure hot reloading, dev server, etc.
