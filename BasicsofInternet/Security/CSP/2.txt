‚úÖ Common Directives         (Rules)
Directive	                  What it controls
default-src           	      Default rule for all content
script-src	                  Allowed JS sources
style-src	                  Allowed CSS
img-src	                      Allowed image URLs
font-src	                  Allowed fonts
connect-src	                  Allowed AJAX / WebSocket / fetch connections
frame-src	                  Allowed iframes
object-src	                  Allowed plugins (like Flash)

üßæ Example 1: Basic Strict CSP
Content-Security-Policy:
  default-src 'self';
  script-src 'self';
  style-src 'self';
üîí Only load HTML, JS, and CSS from your own domain.



üßæ Example 2: Allow Google Fonts, but block everything else

Content-Security-Policy:
  default-src 'self';
  font-src https://fonts.gstatic.com;
  style-src 'self' https://fonts.googleapis.com;


üßæ Example 3: Allow images from CDN and self
Content-Security-Policy:
  img-src 'self' https://cdn.example.com;
